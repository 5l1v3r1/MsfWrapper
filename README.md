MsfWrapper
------
Asynchronously control Metasploit with this library. For example, this will let you run commands in parallel across multiple sessions.

#### Installation
```
git clone https://github.com/DanMcInerney/MsfWrapper
cd MsfWrapper
In a new terminal: msfconsole -r rpc.rc
pipenv install --three
pipenv shell
```

#### Usage
Check out the example script for examples of how to use the API to asynchronously run console commands, session commands, powershell commands, or OS shell commands.
